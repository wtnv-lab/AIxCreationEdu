# Prompts

* [lesson-plan-generation](lesson-plan-generation.md) - 指定した学年・教科・時間数に合わせ、レポートの根拠を示しながら授業案を組み立てるための授業設計支援
* [workshop-design](workshop-design.md) - 学校・自治体・企業研修向けに、半日または1日のワークショップをタイムテーブルや成果物まで整理するための設計支援
* [service-planning](service-planning.md) - レポートの示唆をもとに、生成AI時代の教育サービス案を利用者課題・体験・検証計画まで具体化するためのサービス企画支援
* [idea-generation](idea-generation.md) - 対象者やテーマに応じて、リポジトリ内の知見を横断し、新しい授業・研修・サービス企画の種を提案するためのアイデア創出支援
* [excursion-ideation](excursion-ideation.md) - 固定観念をゆるめ、遠い領域の特徴を借りて、教育実践やサービスのアイデアを広げるための発想支援
* [implementation-roadmap](implementation-roadmap.md) - 授業・研修・企画を実行へ移すために、30日・90日・180日のロードマップと体制・評価・リスクを整理する実装計画支援
* [cross-report-comparison](cross-report-comparison.md) - 指定テーマについて複数レポートを比較し、共通点・差異・相互補完関係・教育実践への示唆を根拠付きで整理する横断比較支援
* [citation-answering](citation-answering.md) - 利用者の質問に対して、レポート本文・メタデータ・図表・参考文献を根拠に、出典付きで回答するための根拠確認支援
* [planning-template](planning-template.md) - 授業・研修・企画を始める前に、目的・対象者・AI活用・評価・ロードマップを空欄形式で整理するための計画立案テンプレート
