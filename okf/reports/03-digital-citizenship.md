# AIを活かしたデジタルシティズンシップ教育

AIの禁止・管理を超え、判断・参加・公共性を育てるデジタルシティズンシップ教育。

# Audience

* 小中高の情報活用能力・情報モラル・ICT活用を担当する教員
* デジタルシティズンシップ教育のカリキュラム設計者
* 生成AI利用ルールや校内研修を設計する学校管理職・ICT担当者
* AIリテラシー教材や安全な学習環境を開発する教育サービス担当者

# Key Takeaways

* 生成AI時代のデジタルシティズンシップは，禁止や管理ではなく，技術社会への判断と参加を学ぶ教育である。
* AI出力を疑い，根拠を確認し，自分の判断として再構成する経験が中核になる。
* 知識を獲得・再編集・共有・表出する過程を，公共的価値へ接続する設計が重要になる。

# Use Cases

* 小中高の情報モラル、ICT活用、AIリテラシー授業
* 学校の生成AI利用ルールや校内研修の設計
* 安全な学習環境を備えた教育サービスの企画
* 生徒の表現活動における倫理・著作権・プライバシー指導

# Learning Activities

* AIの回答について，根拠，偏り，未確認情報をチェックするワークを行う。
* プロンプト，出力，修正過程，利用後の判断を記録し，AI利用を検証可能な学習過程として振り返る。
* 生成物を公開する前に，影響を受ける他者，著作権，個人情報を点検する。

# Implementation Ideas

* 各教科で使えるAI利用チェックリストを作り，授業ごとに更新する。
* 校内研修で，禁止事項だけでなく判断プロセスを共有するケーススタディを行う。
* AI教材に，根拠確認，出典表示，公開前レビュー，振り返り記録の流れを組み込む。
